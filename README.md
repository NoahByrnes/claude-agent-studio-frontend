# Claude Agent Studio - Frontend

React frontend for Claude Agent Studio.

## Tech Stack
- React 18
- TypeScript
- Vite
- TanStack Query
- Radix UI
- Tailwind CSS
- Supabase Auth

## Setup

1. Install dependencies:
```bash
npm install
```

2. Set environment variables (copy from .env.example):
- `VITE_API_BASE`
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

3. Start dev server:
```bash
npm run dev
```

## Deployment

Deploys to Vercel with automatic builds from main branch.
